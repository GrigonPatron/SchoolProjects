**Размер контейнера** :

`docker inspect` не предоставляет прямую информацию о размере контейнера.

Для получения размера контейнера можно использовать команду:

`docker inspect --size --format='{{.SizeRootFs}}' 9a8b99e5512c`

**Cписок замапленных портов** :

`docker inspect --format='{{.NetworkSettings.Ports}}' 9a8b99e5512c`

**ip-адрес контейнера** :

`docker inspect --format='{{.NetworkSettings.IPAddress}}' 9a8b99e5512c`
